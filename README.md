# 📊 Zomato Data Analysis using Power BI

## 🚀 Project Overview

This project focuses on analyzing Zomato restaurant data using Power BI to uncover insights related to pricing, ratings, customer engagement, and restaurant distribution.

## 📁 Dataset

The dataset used in this project is sourced from Kaggle and contains comprehensive information about restaurants listed on Zomato.

It includes features such as:

* Restaurant Name
* Location
* Cuisines
* Ratings
* Cost for two
* Number of Votes

This dataset was used to perform data cleaning, analysis, and visualization to derive meaningful business insights.


## 🧹 Data Cleaning

* Removed missing and inconsistent values
* Converted rating from text to numeric format
* Cleaned cost and votes columns for analysis

## 📊 Dashboard Features

### 🟦 Overview Dashboard

* Total Restaurants
* Average Rating
* Total Votes
* Top Cuisines
* City-wise distribution

### 🟩 Detailed Analysis

* Cost vs Rating (Scatter Plot)
* Rating Distribution
* Votes vs Rating
* Top Restaurants by popularity

### 🟨 Insights

* Most restaurants fall in mid price range
* Ratings mostly between 3.5–4.5
* Higher cost does not guarantee better ratings
* Only a few restaurants receive high engagement

## 🛠️ Tools Used

* Power BI
* Power Query
* Data Visualization

## 💡 Key Learnings

* Data cleaning and transformation
* Dashboard design and visualization
* Extracting business insights from data
